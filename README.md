# No.js
No.js 是一个基于 V8 和 Libuv 的 JS 运行，仅用于学习。

# 开发
1. 获取[依赖库源码](https://pan.quark.cn/s/5ab190a2c430)，复制项目根目录。 
2. 执行 `sh install.sh` 编译依赖库, 成功后会输出 "hello world"。
3. 执行 `./No your_file.js` 运行你的代码。
4. 修改源码后执行 `make` 重新编译代码，然后测试。
